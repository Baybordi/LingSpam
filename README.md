# LingSpam: Spam Email Classifier

## Overview
The **LingSpam** project focuses on classifying emails as spam or not using a **Multinomial Naive Bayes** classifier. This project leverages machine learning techniques to process and classify email datasets effectively. The classifier's performance is evaluated using accuracy, and the model achieves an impressive accuracy score of **97.24%**.

## Dependencies
To run this project, you will need the following Python libraries:

- `scikit-learn`
- `numpy`
- `pandas`
